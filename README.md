# Dream Journey H5 - 造梦江湖1 小宝传奇

H5 remake of the classic 4399 Flash game.

## Play

Open `index.html` in any modern browser. Mobile-friendly (9:16 portrait).

## Tech

- Phaser 3.80.1 (WebGL)
- Single HTML file, zero dependencies
- All game data embedded from original XML

## Features

- Board game movement with dice
- Auto turn-based combat
- Weather system
- 6 chapters, 3 difficulty modes
- 108 enemies, 54 skills, 159 equipment pieces
